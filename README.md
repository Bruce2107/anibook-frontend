# Anibook Frontend

## Frontend para a aplicação [Anibook](https://github.com/Bruce2107/anibook-backend)

### Para iniciar

```sh
yarn start
```

### Testes

```sh
yarn test
```

### ENV

```env
REACT_APP_API_URL=http://localhost:8080
```
